# RentEase PRD

## Goal
RentEase lets customers register, login, browse furniture and appliance rentals, schedule rental delivery, manage active rentals, and request support. Admin users can view inventory and business health.

## Core Requirements
- Registration and login are required before accessing catalog, rentals, checkout, support, admin, or docs.
- Catalog must show furniture and appliances with filter controls.
- Cart must support add, remove, rent total, and security deposit total.
- Checkout must create a rental plan with delivery date, city, location, and tenure.
- Rentals page must show active and historical rental plans.
- Support page must create maintenance, pickup delay, installation, and damage claim requests.
- Admin dashboard must show MRR, open requests, pending deliveries, damage claims, and inventory availability.

## Deliverables
- Functional React web application.
- Node.js and Express.js REST API.
- MongoDB-ready backend with local JSON fallback for preview.
- Admin dashboard.
- PRD and technical documentation.
- Deployment-ready build.
